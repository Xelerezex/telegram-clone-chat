## 1. Базовые входные величины

| Обозначение          | Смысл                                                                       | Значение      | Основание             |
| -------------------- | --------------------------------------------------------------------------- | ------------- | --------------------- |
| $DAU$                | Дневная активная аудитория.                                                 | 17,200,000    | Из раздела 2.         |
| $DMS$                | Отправленных сообщений в день.                                              | 802,724,000   | Из раздела 2.         |
| $DMR$                | Полученных сообщений в день.                                                | 5,160,000,000 | Из раздела 2.         |
| $DSU$                | Сессий на пользователя в день.                                              | 21            | Из раздела 2.         |
| $k_{peak}$           | Коэффициент пика.                                                           | 4             | Из раздела 2.         |
| $MMC$                | Медианное число символов в сообщении.                                       | 136           | Из раздела 2.         |
| $Dialogs\_{session}$ | Среднее число чатов, которые пользователь открывает за одну сессию.         | 2             | Проектное допущение.  |
| $Page$               | Средний размер одной порции истории при открытии чата.                      | 20            | Проектное допущение.  |
| $Dialogs\_{preview}$ | Среднее число диалогов, которые клиент показывает в списке чатов за сессию. | 20            | Проектное допущение.  |
| $k\_{secret\_msg}$   | Доля потока сообщений, приходящаяся на Secret Chats.                        | 5.00%         | Проектное допущение.  |
| $k\_{cloud}$         | Доля потока сообщений, приходящаяся на Cloud Chats.                         | 95.00%        | $1 - k_{secret_msg}$. |

## 2. Производные константы

| Параметр                   | Формула                                                    | Значение   | Смысл                                                          |
| -------------------------- | ---------------------------------------------------------- | ---------- | -------------------------------------------------------------- |
| $Q\_{send}$                | $DMS / 86{,}400$                                           | 9,290.79   | Средний поток отправки сообщений.                              |
| $Q\_{fanout}$              | $DMR / 86{,}400$                                           | 59,722.22  | Средний поток доставок по получателям.                         |
| $Q\_{session}$             | $(DAU \times DSU) / 86{,}400$                              | 4,180.56   | Средний поток сессий и reconnect.                              |
| $Q\_{chat\_open}$          | $(DAU \times DSU \times Dialogs_{session}) / 86{,}400$     | 8,361.11   | Средний поток открытий чатов.                                  |
| $Q\_{history}$             | $Q_{chat_open} \times Page$                                | 167,222.22 | Средний поток чтения истории сообщений.                        |
| $Q\_{preview}$             | $(DAU \times DSU \times Dialogs_{preview}) / 86{,}400$     | 83,611.11  | Средний поток чтения списка чатов и последних сообщений.       |
| $Q\_{send,cloud}$          | $Q_{send} \times k_{cloud}$                                | 8,826.25   | Средний поток записи сообщений в Cloud Chats.                  |
| $Q\_{fanout,cloud}$        | $Q_{fanout} \times k_{cloud}$                              | 56,736.11  | Средний поток доставок по получателям в Cloud Chats.           |
| $Q\_{chat\_open,cloud}$    | $Q_{chat_open} \times k_{cloud}$                           | 7,943.06   | Средний поток открытий Cloud Chats.                            |
| $Q\_{history,cloud}$       | $Q_{history} \times k_{cloud}$                             | 158,861.11 | Средний поток чтения истории Cloud Chats.                      |
| $Q\_{preview,cloud}$       | $Q_{preview} \times k_{cloud}$                             | 79,430.56  | Средний поток чтения списка Cloud Chats и последних сообщений. |
| $Q\_{message\_read,cloud}$ | $Q_{fanout,cloud} + Q_{history,cloud} + Q_{preview,cloud}$ | 295,027.78 | Суммарный поток чтения таблицы `messages`.                     |
| $Q\_{auth}$                | $Q_{send} + Q_{chat_open} + Q_{session}$                   | 21,832.45  | Суммарный поток авторизационных проверок.                      |
| $Q\_{secret\_open}$        | $Q_{chat_open} \times k_{secret_msg}$                      | 418.06     | Средний поток открытий Secret Chats.                           |
| $Q\_{secret\_create}$      | Проектное допущение.                                       | 0.50       | Средний поток создания новых Secret Chats.                     |
| $Fanout$                   | $DMR / DMS$                                                | 6.43       | Среднее число доставок на одну отправку.                       |

| `user_updates`      | $16(id) + 16(user\_id) + 16(chat\_id) + 16(message\_id) + 8(update\_seq\_no) + 16(update\_kind\ avg) + 8(created\_at) \approx 80$. Округление вверх.                                                              | 96 Б            |
| `device_sync_state` | $16(user\_id) + 24(device\_id\ avg) + 8(last\_update\_seq\_no) + 8(last\_sync\_at) = 56$. Округление вверх.                                                                                                       | 64 Б            |
| `user_presence`     | $16(user\_id) + 24(device\_id\ avg) + 8(status\ avg) + 8(last\_seen\_at) + 8(updated\_at) = 64$.                                                                                                                  | 64 Б            |

## 5. Источник метрик

| Число или группа чисел                                                                                                                                                                     | Статус                                               | Комментарий                                                                                                                                                                                                  |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| $DAU$, $DMS$, $DMR$, $DSU$, $k_{peak}$, $MMC$                                                                                                                                              | Исходные данные.                                     | Взяты из раздела 2.                                                                                                                                                                                          |
| $Q_{send}$, $Q_{fanout}$, $Q_{session}$                                                                                                                                                    | Строго посчитано.                                    | Выводятся напрямую из исходных метрик.                                                                                                                                                                       |
| $Q_{chat_open}$, $Q_{history}$, $Q_{preview}$, $Q_{send,cloud}$, $Q_{fanout,cloud}$, $Q_{chat_open,cloud}$, $Q_{history,cloud}$, $Q_{preview,cloud}$, $Q_{message_read,cloud}$, $Q_{auth}$ | Строго посчитано после фиксации проектных допущений. | Эти величины нужны, чтобы правильно посчитать чтение истории сообщений, чтение списка чатов и работу авторизации.                                                                                            |
| Размер `messages = 640` Б                                                                                                                                                                  | Строго посчитано после фиксации структуры строки.    | Таблица хранит тело сообщения, потому что Cloud Chats в Telegram не являются end-to-end encrypted. ([Telegram](https://core.telegram.org/techfaq?utm_source=chatgpt.com "FAQ for the Technically Inclined")) |
| Размеры остальных таблиц                                                                                                                                                                   | Инженерная оценка.                                   | Это логический размер записи без page overhead и индексов.                                                                                                                                                   |
| `messages.read = 295,027.78`                                                                                                                                                               | Основной результат перерасчета.                      | Именно эта величина делает `messages` главной таблицей по чтению.                                                                                                                                            |
| `secret_chats`                                                                                                                                                                             | Таблица только метаданных.                           | Она описывает Secret Chats, но не хранит серверную историю их сообщений. ([Telegram](https://core.telegram.org/api/end-to-end?utm_source=chatgpt.com "End-to-End Encryption, Secret Chats"))                 |
| $Dialogs\_{session}$, $Page$, $Dialogs\_{preview}$, $k\_{secret\_msg}$, $k\_{cloud}$                                                                                                       | Проектные допущения.                                 | Добавлены, чтобы отделить Secret Chats от Cloud Chats в расчетах серверной БД и корректно оценить реальную read-нагрузку на `messages`.                                                                      |
