## 1. Базовые входные величины

|Обозначение|Смысл|Значение|Откуда взято|
|---|---|---|---|
|$DAU$|Дневная активная аудитория.|17,200,000|Из раздела 2.|
|$DMS$|Отправленных сообщений в день.|802,724,000|Из раздела 2.|
|$DMR$|Полученных сообщений в день.|5,160,000,000|Из раздела 2.|
|$DSU$|Сессий на пользователя в день.|21|Из раздела 2.|
|$Batch$|Средний batch sync/update.|10|Из раздела 2.|
|$k_{peak}$|Коэффициент пика.|4|Из раздела 2.|

## 2. Расчет QPS

| Параметр      | Смысл                                    | Значение  | Формула                                   |
| ------------- | ---------------------------------------- | --------- | ----------------------------------------- |
| $Q_{send}$    | Средний поток отправки сообщений.        | 9,290.79  | $DMS / 86{,}400$                          |
| $Q_{fanout}$  | Средний поток доставок по получателям.   | 59,722.22 | $DMR / 86{,}400$                          |
| $Q_{session}$ | Средний поток сессий и reconnect.        | 4,180.56  | $(DAU \times DSU) / 86{,}400$             |
| $Q_{sync}$    | Средний поток sync/getUpdates запросов.  | 5,972.22  | $DMR / (Batch \times 86{,}400)$           |
| $Q_{event}$   | Суммарный поток событий.                 | 73,193.56 | $(DMS + DMR + DAU \times DSU) / 86{,}400$ |
| $Fanout$      | Среднее число доставок на одну отправку. | 6.43      | $DMR / DMS$                               |

## 5. Источник метрик

| Число или группа чисел                                                                                                                         | Статус              | Комментарий                                                                   |
| ---------------------------------------------------------------------------------------------------------------------------------------------- | ------------------- | ----------------------------------------------------------------------------- |
| $DAU$, $DMS$, $DMR$, $DSU$, $Batch$, $k_{peak}$, $MMC$                                                                                         | Исходные данные.    | Взяты из раздела 2.                                                           |
| $Q_{send}$, $Q_{fanout}$, $Q_{session}$, $Q_{sync}$, $Q_{event}$, $Fanout$                                                                     | Строго посчитано.   | Выводятся напрямую из исходных метрик.                                        |
| $\text{messages.body} = 544$ Б                                                                                                                 | Строго посчитано.   | $136 \times 4$.                                                               |
| `user_sessions.write`, `chats.write`, `chat_members.write`, `messages.write`, `update_buffer.write`, `presence_cache.write`, `event_log.write` | Строго посчитано.   | Выводятся из основных событий продукта.                                       |
| Размеры строк кроме $\text{messages.body}$                                                                                                     | Строго посчитано.   | Проектный логический размер записи.                                           |
| $\text{users.write} = 1$ QPS                                                                                                                   | Допущение.          | Редкое изменение профиля.                                                     |
| $\text{users.read} = Q_{session}$                                                                                                              | Допущение.          | Профиль читается на session-start или reconnect.                              |
| $\text{messages.read} = Q_{sync}$                                                                                                              | Допущение.          | В базовой модели оценивается минимальный online-path без тяжелого скроллинга. |
| $\text{event\_log.read} = 0$                                                                                                                   | Ограничение модели. | Аналитическое чтение вынесено за online-контур.                               |
